# Requirements

- The program must be able to take a number and return its multiplicative persistence.

# Acceptance Criteria

- The program must be able to take an integer as input.
- The program must be capable of returning another integer. 

# Scenarios

- 5 => 0
- 54 => 2
- 796 => 5
- -814 => 2
- 44168495 => 2
